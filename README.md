## 📄 Descripción
**Coder Flex App** es una plataforma de comercio electrónico diseñada para brindar una experiencia de compra fluida. La aplicación permite visualizar un catálogo de productos, filtrar por categorías, ver detalles específicos y gestionar un carrito de compras. La persistencia de datos y órdenes de compra se maneja en la nube mediante Firebase.

## ✨ Características (Features)
* **Navegación SPA:** Transiciones rápidas sin recarga de página gracias a *React Router DOM v7*.
* **Gestión de Formularios:** Manejo eficiente de datos de contacto y checkout mediante *React Hook Form*.
* **Feedback al Usuario:** Notificaciones visuales estilizadas con *SweetAlert2* y estados de carga con *React Spinners*.
* **Base de Datos:** Catálogo y órdenes sincronizados con *Firebase Firestore*.
* **Diseño Responsivo:** Interfaz adaptable a diferentes dispositivos.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido utilizando las siguientes librerías y herramientas:

### Core & Build
* **[React JS](https://react.dev/):** Librería principal para la interfaz de usuario.
* **[Vite](https://vitejs.dev/):** Entorno de desarrollo y empaquetador de próxima generación.

### Navegación & Datos
* **[React Router Dom](https://reactrouter.com/):** Manejo de rutas.
* **[Firebase](https://firebase.google.com/):** Backend-as-a-Service (BaaS) para base de datos.

### UI & UX
* **[React Icons](https://react-icons.github.io/react-icons/):** Paquete de iconos.
* **[SweetAlert2](https://sweetalert2.github.io/):** Alertas personalizadas y pop-ups.
* **[React Spinners](https://www.npmjs.com/package/react-spinners):** Indicadores de carga.
* **[React Hook Form](https://react-hook-form.com/):** Validación y manejo de formularios.

## 🚀 Instalación y Puesta en Marcha

Sigue estos pasos para clonar y ejecutar el proyecto localmente:

1.  **Clonar el repositorio:**
    ```bash
    git clone <URL_DE_TU_REPOSITORIO>
    ```

2.  **Ingresar al directorio:**
    ```bash
    cd proyecto-coder
    ```

3.  **Instalar dependencias:**
    ```bash
    npm install
    ```

4.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```
    > La aplicación correrá generalmente en `http://localhost:5173`.

## 📦 Scripts Disponibles

* `npm run dev`: Corre la app en modo desarrollo.
* `npm run build`: Compila la app para producción.
* `npm run lint`: Ejecuta ESLint para buscar errores de código.
* `npm run preview`: Sirve la versión compilada localmente para probarla.
